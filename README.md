# Python Programming Repository

## Description

This repository contains Python programs covering the fundamental concepts of Python programming as part of my academic syllabus and self-practice.
It includes examples, exercises, and test cases to strengthen problem-solving and programming skills.

## Topics Covered

### 1. Variables and Data Types

* Integer, Float, String, Boolean
* Type conversion
* Input and output operations
* Keywords and Identifiers
* Operators(Arithmetic,Relational,Logical,Bitwise,etc)

### 2. Looping Statements

* for loop
* while loop
* Nested loops
* Loop control statements (break, continue, pass)

### 3. Functions in Python

* Defining functions
* Function parameters and return values
* Built-in functions
* User-defined functions
* Function test cases

### 4. Error Types and Exception Handling

* Syntax errors
* Runtime errors
* Logical errors
* try and except blocks
* Error handling techniques

### 5. Data Structures in Python

* Lists
* Tuples
* Sets
* Dictionaries

### 6. Assignments and Practice Programs

* Basic programming exercises
* Problem-solving assignments
* Logic-building programs


## Technologies Used

* Spyder
* Visual Studio Code / IDLE

## Purpose of This Repository

The purpose of this repository is to:

* Learn and practice Python programming concepts
* Build a strong foundation in programming
